# Image Converter

A tool to bring any image to the catppuccin palette

Transform each wallpaper, icon, image into the smoothest palette on the internet

Aplicación simple con interfaz gráfica para convertir imágenes usando una paleta de colores personalizada.

## Características

- Interfaz moderna con CustomTkinter
- Vista previa antes y después
- Conversión usando paleta de colores Catppuccin
- Fácil de usar

## Requisitos

```bash
pip install customtkinter
pip install ImageGoNord
pip install Pillow
```

## Uso

1. Ejecuta la aplicación:
```bash
python image_converter_gui.py
```

2. Haz clic en "Seleccionar Imagen"
3. Haz clic en "Convertir Imagen"
4. La imagen convertida se guarda como `output.jpg`

## Paleta de Colores

La aplicación usa una paleta Catppuccin con los siguientes colores:
- Base: `#1e1e2e`, `#313244`
- Texto: `#cdd6f4`
- Acentos: `#f38ba8`, `#a6e3a1`, `#89b4fa`, `#f9e2af`, `#cba6f7`
